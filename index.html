Projects

Below are a few recent projects that I've worked on:

7.CSS

7.CSS is a CSS framework for building Windows 7 graphical user interface elements, i.e. buttons, checkboxes, windows, etc.

At that time, I didn't know much about SASS but after a quick research and some experimentations, I started making some contributions to the repo.

Some of my contributions were: replicating the reflection effect on the window, adding animations for the push button, window control buttons, and more. Well, it's mostly changes to make the visual looks as close as possible to the originals.

In case you can't tell, yes, it's the framework that I'm using to style the windows for this page.

Win7 Simu

Back in around November of 2023, I was so lucky to be invited to work on this project. I accepted without hesitation because I LOVE Windows 7 and this is an interesting project.

Win7 Simu is a simulator with the ideas of replicating the looks and feels of the Windows 7 operating system for entertainment purposes and bringing back the nostalgia.

At first, my job was creating themes and adjusting the visual appearance to best match the original Windows 7. But now I'm also developing apps for Win7 Simu, e.g. I'm currently working on replicating the famous Windows Media Centre (WMC) that many people knew and loved. It was such a wild and fun experience working on this because I didn't have any prior knowledge about Vue.js which is the tech that Win7 Simu uses. At the end, I still managed to learn and complete all the basic functionalities for WMC on Win7 Simu and I'm still working on completing it as of the time of this writing.

Below is the quick demo video of WMC in Win7 Simu. Or you can experience it for yourself at https://win7simu.visnalize.com.
